# Offboarding Checklists

Comprehensive task lists for smooth employee exits.

---

## Universal Offboarding Checklist

Use this as base template for all departures.

### Pre-Departure

**Manager:**
- [ ] Accept resignation or deliver termination decision
- [ ] Notify HR immediately
- [ ] Create transition plan
- [ ] Schedule exit interview
- [ ] Announce to team (within 24 hours)
- [ ] Identify who will take over responsibilities

**HR:**
- [ ] Create exit paperwork
- [ ] Calculate final paycheck and PTO payout
- [ ] Prepare COBRA notification
- [ ] Update HRIS system
- [ ] Schedule exit interview (if not done by manager)
- [ ] Send offboarding checklist to manager and employee

**IT:**
- [ ] Audit all access and systems
- [ ] Plan access revocation timeline
- [ ] Prepare equipment return process

---

### Week 1: Knowledge Transfer Begins

**Employee:**
- [ ] Document all active projects and status
- [ ] Create handoff documentation
- [ ] List all systems and access they have
- [ ] Identify critical passwords/credentials
- [ ] Schedule training sessions with team

**Manager:**
- [ ] Review handoff documentation
- [ ] Assign interim owners for responsibilities
- [ ] Monitor knowledge transfer progress
- [ ] Daily check-ins

**Team:**
- [ ] Shadow departing employee
- [ ] Ask questions and take notes
- [ ] Begin taking over responsibilities

---

### Week 2: Finalize Transition

**Employee:**
- [ ] Complete all handoff documentation
- [ ] Finish training team members
- [ ] Return company property
- [ ] Complete exit interview
- [ ] Clean out desk/workspace

**Manager:**
- [ ] Verify all handoffs complete
- [ ] Conduct exit interview (or ensure HR did)
- [ ] Write LinkedIn recommendation (if appropriate)
- [ ] Send thank-you message

**HR:**
- [ ] Process final paycheck
- [ ] Send COBRA notification
- [ ] Provide 401(k) rollover information
- [ ] Collect signed exit paperwork

**IT:**
- [ ] Collect all equipment
- [ ] Revoke all system access (end of last day)
- [ ] Forward email to manager (7 days)
- [ ] Transfer file ownership
- [ ] Remove from distribution lists

---

### Post-Departure (Within 1 Week)

**HR:**
- [ ] Analyze exit interview data
- [ ] Update turnover metrics
- [ ] File exit paperwork
- [ ] Send alumni network invitation

**Manager:**
- [ ] Update org chart
- [ ] Redistribute responsibilities
- [ ] Begin backfill process (if needed)
- [ ] Update team documentation

**IT:**
- [ ] Permanently delete access (after 30 days)
- [ ] Archive employee files

## Role-Specific Checklists

- [ ] Share deployment procedures
- [ ] Handoff on-call responsibilities

## Scenario-Specific Checklists

### Voluntary Resignation (Good Terms)

**Additional steps:**
- [ ] Offer to be a reference
- [ ] Write LinkedIn recommendation
- [ ] Invite to alumni network
- [ ] Stay connected personally
- [ ] Consider boomerang opportunity in future

**Tone:** Warm, appreciative, open door

---

### Voluntary Resignation (Performance Issues)

**Additional steps:**
- [ ] Keep exit interview brief and factual
- [ ] Don't make reference promises
- [ ] Revoke access promptly (but not early)
- [ ] Monitor knowledge transfer closely

**Tone:** Professional, neutral

---

### Termination (Performance)

**Additional steps:**
- [ ] Consult HR and Legal before termination meeting
- [ ] Have witness present in termination meeting
- [ ] Immediate access revocation (same day)
- [ ] Escort to collect personal belongings (if necessary)
- [ ] Provide severance details in writing
- [ ] Skip or brief exit interview
- [ ] Monitor for security risks

**Tone:** Direct, respectful, final

**Legal:** Document all performance issues thoroughly. Consult employment lawyer.

---

### Layoff (RIF)

**Additional steps:**
- [ ] Provide severance package details
- [ ] Offer outplacement services
- [ ] Write LinkedIn recommendations
- [ ] Make introductions to network
- [ ] Consider consulting arrangements
- [ ] Maintain relationship (not performance-based)

**Tone:** Compassionate, supportive, helpful

**Communication:** Be transparent about business reasons. Don't sugarcoat.

---

### Retirement

**Additional steps:**
- [ ] Celebrate contributions (party, gift, etc.)
- [ ] Extended knowledge transfer timeline (months if possible)
- [ ] Consider consulting arrangement
- [ ] Document institutional knowledge
- [ ] Maintain relationship

**Tone:** Celebratory, grateful, honoring

## Access Revocation Checklist

**Critical (Revoke EOD on last day):**
- [ ] Email (forward to manager for 7 days)
- [ ] Slack
- [ ] VPN
- [ ] Cloud services (AWS, GCP, Azure)
- [ ] GitHub/code repositories
- [ ] Production databases
- [ ] Admin panels

**Secondary (Revoke within 24 hours):**
- [ ] Zoom, Google Meet
- [ ] Notion, Confluence, wiki
- [ ] Design tools (Figma)
- [ ] Analytics platforms
- [ ] Marketing tools
- [ ] CRM (Salesforce)

**Remove from:**
- [ ] Distribution lists
- [ ] Slack channels
- [ ] Google Groups
- [ ] Shared calendars
- [ ] Team directories

## HR Paperwork Checklist

**Employee must sign:**
- [ ] Resignation letter (if voluntary)
- [ ] Acknowledgment of final paycheck details
- [ ] COBRA election form (if applicable)
- [ ] NDA/confidentiality reminder
- [ ] Return of company property form
- [ ] Non-compete reminder (if applicable)

**HR must provide:**
- [ ] Final paycheck details (amount, date)
- [ ] Accrued PTO payout
- [ ] COBRA notification and forms
- [ ] 401(k) rollover information
- [ ] Unemployment information (if applicable)
- [ ] Reference policy

---

## Manager's Final Day Checklist

**Morning:**
- [ ] Personal goodbye (1:1, 15-30 min)
- [ ] Team meeting or standup (if appropriate)
- [ ] Time for employee to say goodbye to teammates

**Afternoon:**
- [ ] Equipment return to IT
- [ ] Final paperwork with HR
- [ ] Thank-you message to employee

**End of Day:**
- [ ] Coordinate access revocation with IT
- [ ] Send team message thanking employee
- [ ] Personal thank-you note (email or handwritten)

**Next Day:**
- [ ] Update team on transition plan
- [ ] Redistribute responsibilities
- [ ] Update documentation and org chart

## Timeline Templates

### 2-Week Notice (Standard)
- **Day 1:** Accept resignation, notify HR, plan transition
- **Day 1-2:** Announce to team
- **Week 1:** Knowledge transfer begins
- **Week 2:** Finalize handoffs
- **Last day:** Equipment return, access revoked
- **Week 3:** Alumni invitation, analyze exit data

### 30-Day Notice (Senior/Critical Role)
- **Week 1:** Transition planning, announcements
- **Week 2-3:** Intensive knowledge transfer
- **Week 4:** Wrap-up and final handoffs
- **Post-departure:** Alumni network, backfill planning

### Immediate Termination
- **Day 1:** Termination meeting, immediate access revocation
- **Day 1:** Equipment collection
- **Week 1:** Knowledge recovery (from documentation, team)
- **Week 1:** Final paycheck and paperwork

## Post-Exit Analysis Checklist

**HR & Leadership:**
- [ ] Review exit interview data
- [ ] Identify patterns (manager issues, comp, culture)
- [ ] Update turnover metrics
- [ ] Share insights with leadership (quarterly)
- [ ] Update offboarding process based on feedback

**Manager:**
- [ ] Reflect on what could have retained them
- [ ] Adjust team processes if needed
- [ ] Plan backfill or redistribution
- [ ] Update documentation

---

## Success Metrics

Track these to improve offboarding:
- Exit interview completion rate (goal: 95%+)
- Knowledge transfer completeness (manager rating)
- Equipment return rate (goal: 100%)
- Access revocation timeliness (goal: same-day)
- Alumni network join rate (goal: 60%+)
- Boomerang hire rate
- Time to backfill role

Great offboarding = smooth transition + preserved knowledge + maintained relationships.
